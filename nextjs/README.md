
# Next.js Prompts

Bu klasör, Next.js projelerinde kullanabileceğin ChatGPT prompt örneklerini içerir.  
Amacı, sık kullanılan geliştirme senaryolarında hızlı çözüm ve öneri almak.

## Kullanım
1. `PROMPTS.md` dosyasındaki promptlardan ihtiyacını kopyala.
2. ChatGPT’de kullan veya kendi otomasyon araçlarına ekle.
3. Yeni prompt ekleyerek klasörü güncel tutabilirsin.
